# 依据

Use this file when the user asks why a rule exists or when final advice depends on
current reporting guidelines.

## Source hierarchy

1. Target journal instructions and current author guidelines.
2. Official reporting guideline pages and original guideline papers.
3. Domain standards such as IBSI for radiomics feature definitions.
4. Peer-reviewed methodological papers and systematic reviews.
