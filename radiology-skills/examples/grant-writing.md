# Example: grant proposal polishing

## Input

```text
帮我把这段省自然立项依据改得更适合申报书：我们准备用 CT 影像组学预测肺癌免疫治疗疗效，目前模型 AUC 0.82，希望提高预测准确率。
```

## Expected output shape

```text
基金申报定位
- 项目类型：省级自然科学基金
- 适合强调的主线：影像可量化表型与免疫治疗应答异质性的关系
- 当前最弱环节：科学问题不够清楚，前期基础和验证设计不足

原文问题诊断
- “提高准确率”偏工程目标，需要转化为可资助的科学问题。

结构补强建议
- 补充免疫治疗疗效差异的临床痛点。
- 明确影像组学特征与治疗应答的生物学联系。
- 说明样本量、标签来源、外部/时间验证和校准计划。

润色稿
[在不编造前期结果的前提下，改写为基金式段落]
```
