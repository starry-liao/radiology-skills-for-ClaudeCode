# Fit grading — score the paper, then build the ladder

Grade the manuscript on the dimensions that decide tier, name the limiting factor, then map to
reach/target/safety.

## Grade these dimensions

| Dimension | Weak | Strong |
|---|---|---|
| **Validation** | Internal CV only | Fully external / prospective |
| **Centers** | Single | Multi-center with per-center results |
| **Prospectivity** | Retrospective | Prospective / temporal |
| **Clinical utility** | AUC only | Calibration + decision-curve + reader study |
| **Sample size / events** | Small / few events | Adequate for the claim |
| **Novelty** | Incremental | Clear, defensible, possibly mechanistic |
| **Reporting** | Gaps | CLAIM/CLEAR/TRIPOD+AI/IBSI compliant |

Score each (e.g. weak / moderate / strong) and **name the single limiting dimension** — it sets
the ceiling.

## Map grade → ladder

- **Reach** — one tier above the honest grade; justified only if a standout strength (e.g.
  prospective multi-center) offsets the weakness. State the risk.
- **Target** — the tier the grade matches; the realistic best fit.
- **Safety** — a solid venue likely to accept with minor revision; protects against a long cascade.

For each rung: **venue (verify live) · match reason · risk · turnaround consideration · the one
thing to strengthen to climb.**

## Move-up levers (highest leverage first)

1. Add **external validation** (single biggest lever) → radiology-design.
2. Add a **reader study / clinical-utility** analysis → radiology-translation.
3. Add **calibration + decision-curve** → radiology-stats.
4. Add **biological grounding** → radiology-radiogenomics.
5. Close **reporting-guideline** gaps → radiology-reporting.

## Output skeleton

```
Paper profile:   [type | selling point | biggest weakness]
Fit grade:       [per-dimension + limiting factor]
Ladder:
  Reach:   [venue* | reason | risk | strengthen-to-climb]
  Target:  [venue* | reason | risk]
  Safety:  [venue* | reason]
Strengthen-first: [ordered levers]
Verify now:      [live scope checks → radiology-search]
(* current scope to be verified live)
```
